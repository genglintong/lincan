# Java 练手项目

这是一个Java学习练手项目，用于整理和管理Java编程练习。

## 项目结构

```
lincan/
├── run_project.sh          # 项目运行脚本
├── README.md              # 项目简介
└── YYYYMMDD/              # 按日期组织的练习目录
    ├── README.md          # 当日练习详细说明
    └── 各个小项目/          # 具体的练习项目
```

## 快速开始

1. 查看可用的练习项目：
```bash
./run_project.sh --help
```

2. 运行指定日期的项目：
```bash
./run_project.sh <日期> <项目名称>
```

## 添加新练习

当需要添加新的练习时：
1. 创建新的日期目录：`mkdir YYYYMMDD`
2. 在日期目录下创建功能分类的子目录
3. 将Java文件放入对应的子目录
4. 更新运行脚本（如需要）

各日期目录下的README.md包含该日练习的详细说明和功能介绍。
